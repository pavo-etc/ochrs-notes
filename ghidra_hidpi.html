Making Ghidra scale on HiDPI screens.

Tested on Kali Linux 5.18.0-kali5-amd64.

Ghidra doesn't scale to HiDPI screens, but this can be fixed by modifying the launch.properties file. This is located in /support/launch.properties, which when installed using apt is /usr/share/ghidra/support/launch.properties.

Change this setting:

VMARGS_LINUX=-Dsun.java2d.uiScale=1

to:

VMARGS_LINUX=-Dsun.java2d.uiScale=2
